LOG540.TP1
==========
This project use the [HalfNES](https://code.google.com/p/halfnes/) source code and adds a network functionnality to play with one of your friend. This was done for the first laboratory in LOG540 (Analyse et conception de logiciels de télécommunications) at the École de technologie supérieure.

Requirements
------------
- Java 7
- [NES Rom] (http://www.dpadhero.com/Home.html)

Installation
------------
Run the .jar file to start the emulator and open a ROM to play.
